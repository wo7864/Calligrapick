# Celligrapick
GAN을 활용한 손글씨 생성 프로젝트입니다.
현재는 프로토 타입으로, 영어 손글씨를 생성할 수 있습니다.
최종적으로는 한글 캘리그라피를 생성하는 것을 목표로 합니다.
